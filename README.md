# Colombian Instruments
XML definitions for MuseScore of typical colombian instruments.

## Instruments

### Plucked string instruments

All plucked string instruments have tablature functionality included.

* Bandola Andina
  * Soprano
  * Soprano Bb
  * Alto 
  * Bajo (Baritono)
* Bandola Llanera
* Cuatro
* Guitarra Requinto
* Tiple
