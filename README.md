# colombian-strings-musescore
XML definitions for MuseScore of typical Colombian plucked string instruments.

# Instruments
All instruments have tablature functionality included.

* Bandola Andina
* Bandola Andina Bb
* Bandola Alto 
* Bandola Bajo (Baritone)
* Bandola Llanera
* Cuatro
* Guitarra Requinto
* Tiple
